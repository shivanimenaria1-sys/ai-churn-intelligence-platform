# Model Artifacts

Trained models and preprocessing pipelines are generated by the notebooks and **not committed** to Git (see root `.gitignore`).

## Generate artifacts

Run in order from the project root:

```bash
jupyter notebook notebooks/feature_engineering.ipynb
jupyter notebook notebooks/model_training.ipynb
```

## Required for API & dashboard

| File | Produced by |
|------|-------------|
| `churn_xgboost_model.joblib` | `model_training.ipynb` |
| `artifacts/churn_preprocessor.joblib` | `feature_engineering.ipynb` |
| `artifacts/feature_names.joblib` | `feature_engineering.ipynb` |

## Optional (notebooks & SHAP)

| File | Purpose |
|------|---------|
| `artifacts/X_train.joblib` | Training features (processed) |
| `artifacts/X_test.joblib` | Test features — used for SHAP in dashboard |
| `artifacts/y_train.joblib` | Training labels |
| `artifacts/y_test.joblib` | Test labels |

## Layout

```
models/
├── churn_xgboost_model.joblib    # Trained XGBoost classifier
└── artifacts/
    ├── churn_preprocessor.joblib
    ├── feature_names.joblib
    ├── X_train.joblib
    ├── X_test.joblib
    ├── y_train.joblib
    └── y_test.joblib
```
